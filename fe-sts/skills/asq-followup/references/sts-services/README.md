# STS Services Catalog

This directory contains detailed information about Shared Technical Services (STS) offerings extracted from the STS Services presentation.

## Service Categories

### 🏗️ Platform Services
Core platform setup and configuration services for establishing robust Databricks environments.

- **[Workspace Setup](./platform/workspace-setup/)** - Initial workspace provisioning and configuration
- **[Unity Catalog Setup](./platform/unity-catalog-setup/)** - Unified governance implementation
- **[CI/CD](./platform/cicd/)** - Continuous integration and deployment pipelines

### 📊 Data Engineering
Services for building scalable, reliable data pipelines and optimizing data processing.

- **[Lakeflow (DLT)](./data-engineering/lakeflow/)** - Delta Live Tables implementation
- **[Delta Optimizations](./data-engineering/delta-optimizations/)** - Performance and cost optimization
- **[Data Sharing](./data-engineering/data-sharing/)** - Delta Sharing implementation

### 📈 Data Warehousing
Analytics and business intelligence services for data democratization.

- **[Genie Foundation](./data-warehousing/genie-foundation/)** - Natural language querying setup
- **[DBSQL](./data-warehousing/dbsql/)** - SQL warehouse optimization
- **[AI/BI Dashboards](./data-warehousing/ai-bi/)** - AI-powered dashboard development

### 🤖 ML & GenAI
Machine learning and generative AI implementation services.

- **[AI/ML Pipelines](./ml-genai/ai-ml-pipelines/)** - End-to-end ML pipeline development
- **[Gen AI Apps](./ml-genai/gen-ai-apps/)** - RAG and GenAI application development
- **[MLOps](./ml-genai/mlops/)** - ML operations and lifecycle management

### 🔄 Migration Services
Tools and services for migrating from legacy platforms to Databricks.

- **[Lakebridge Analyzer](./migration/lakebridge-analyzer/)** - Migration assessment and planning
- **[Lakebridge Transpiler](./migration/lakebridge-transpiler/)** - Automated code translation

## Service Structure

Each service directory contains:
- `README.md` - Detailed service description including:
  - Overview and key activities
  - Benefits and target audience
  - Prerequisites and deliverables
  - Typical duration and success metrics

## Quick Reference

| Service Category | Service Count | Typical Duration |
|-----------------|---------------|------------------|
| Platform | 3 services | 1-4 weeks each |
| Data Engineering | 3 services | 1-4 weeks each |
| Data Warehousing | 3 services | 1-3 weeks each |
| ML & GenAI | 3 services | 3-8 weeks each |
| Migration | 2 services | 2-3 weeks assessment |

## How to Use This Catalog

1. **Identify Your Need** - Browse categories to find relevant services
2. **Review Prerequisites** - Check the prerequisites section in each service README
3. **Understand Deliverables** - Review what will be delivered
4. **Estimate Timeline** - Use the duration estimates for planning
5. **Check Success Metrics** - Understand how success is measured
